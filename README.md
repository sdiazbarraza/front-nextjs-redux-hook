# Basic example of the nextjs+react+redux+hooks+antd

_This project is app frontend basic_
### Prerequiste
```
 -Node
```
### Clone
```
 git clone git@github.com:simp06/front-nextjs-redux-hook.git <folder>
```
### How to run

```
 npm install
 npm run build
 npm run start
```

### Go  frontend
_In the browser_
```
localhost:3000
```
